# U-Haul QA proejct

## Basic commands
Cypress install command
```
npm install cypress --save-dev 
```
### QA
To execute the script straight away from the command line
 ```
 npx cypress run
 ```
 Runs ui cypress
 ```
 npx cypress open
 ```
